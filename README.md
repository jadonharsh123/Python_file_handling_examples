# 📂 Python File Handling Examples

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Status](https://img.shields.io/badge/Status-Learning%20Project-green)

This repository contains **Python programs demonstrating file handling operations** such as reading, writing, appending, and processing text and binary files.

The project also includes examples of **Python modules, package imports, and binary file handling using pickle**.

---

# 🚀 Features

* 📄 Read entire text file
* 📑 Read first **N lines** from a file
* ✏️ Append text to an existing file
* 🔁 Reverse lines of a file
* 📋 Store file data in a list
* 📊 Count **characters, words, and lines**
* 📦 Binary file handling using **pickle**
* 🧩 Custom Python module (`mymod.py`)
* 📂 Package import example
